# Decisions

### milestone_structure
- **Decision:** create_new
- **Reasoning:** The approved exhibition structure is a strong fit for the expected lifecycle.
- **Risk:** Low risk if reused as-is.

### sop
- **Decision:** adapt_existing
- **Reasoning:** The exhibition SOP is relevant, but the compressed timeline may require adjusted sequencing.
- **Risk:** Unadapted SOP guidance could leave setup tasks misordered.

### production_calendar
- **Decision:** create_new
- **Reasoning:** No project-specific calendar exists yet; creating one from the exhibition baseline is the safest path.
- **Risk:** Without a calendar, the project cannot be activated cleanly.

### drive_setup
- **Decision:** create_new
- **Reasoning:** No project Drive structure has been found, so the system should create the standard project folder layout.
- **Risk:** Missing Drive structure will block file-based setup and references.

### contract_setup
- **Decision:** adapt_existing
- **Reasoning:** A standard contract template exists, but venue-specific terms may need revision.
- **Risk:** Using the template unchanged could miss partner obligations.
