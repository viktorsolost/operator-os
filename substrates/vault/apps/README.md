# Apps

This folder holds app-level product context related to systems built around VIK OS.

Use this folder for:
- app-specific summaries
- product framing
- app-specific decisions
- app-specific open questions
- archives that belong to one app

Do not use this folder for:
- canonical boot authority
- global operator rules
- raw runtime state from Memento
- active project folders that belong in `../projects/`

Rule:
If the work is about one product or app as a product surface, it can live here.
If the work is about a client, show, partnership, or execution project, it belongs in `../projects/` or `../intake/`.
