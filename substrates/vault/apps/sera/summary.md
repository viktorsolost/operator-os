# Sera

## What It Is

Sera is the system.

It is the app / operating environment that connects a person to:

- live company systems
- project context
- durable memory
- controlled agents
- next actions

The dashboard is only one surface inside Sera. It is not the whole product.

## Core Idea

One person uses their own app on their own computer.

That app:

- syncs shared company truth
- keeps local operational state
- stores durable memory in markdown
- lets agents help with context, drafts, decisions, and actions
