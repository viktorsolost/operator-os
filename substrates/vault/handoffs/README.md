# Handoffs

This folder holds explicit cross-operator handoff artifacts.

Use this folder for:
- handoff packets from one operator to another
- bounded transition notes when ownership changes
- historical handoff records worth keeping

Do not use this folder for:
- hidden authority
- always-loaded startup context
- project summaries

Rule:
A handoff is a transfer artifact, not a source of silent ongoing authority.
If a handoff creates a durable truth, that truth should be moved into the proper canonical file or project context.
