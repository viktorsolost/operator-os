# Eterno Metrics Framework

Status: draft

## Rule

Every doctrine claim should eventually be checked against evidence.
Every channel should have clear metrics.
Every iteration should leave a measurable trace.

## Core metrics to track

### X
- posts per week
- impressions
- engagement rate
- profile visits
- link clicks
- follower change
- reach multiplier on active vs silent days

### Instagram
- posts per week
- reach
- saves
- shares
- profile visits
- link clicks
- follower change

### Newsletter
- sends
- open rate
- click-through rate
- click-to-open rate
- unsubscribe rate
- revenue or traffic attributed

### Website / Shopify
- sessions
- source / medium
- product page views
- add to cart
- conversion rate
- revenue by channel
- bounce / exit on key pages

### Cross-channel discipline
- UTM coverage rate
- time from campaign launch to first review
- number of experiments run
- number of learnings captured

## Review principle

Eterno should not optimize from opinion alone.
The loop is:
ship, measure, compare, question, improve.
